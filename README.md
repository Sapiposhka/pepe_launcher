
Данный лаунчер явлется портативным, а следствено не нужна установка и права администратора, которых иногда может не быть
Игры, которые содержатся в лаунчере тоже являются портативными(почти все, но не одна из не требует установки и прав администратора)

Все работает через облачное хранилище Яндекс Диск (слава Яндексу), где хранятся игры, превьюшки для них и сам лаунчер в зип формате 
Но для полноценной работы еще используются json файлы:
1. список всех игр (https://disk.yandex.ru/d/Xues-OFVC0luIw)
2. манифест обновлений (https://disk.yandex.ru/d/qm_LJch1bPrHfQ)

Инструкция:
Открываем проект в Visual Studio 2022 и публикуем проект Pepe_Launcher, который находится в обозреватиле решений
