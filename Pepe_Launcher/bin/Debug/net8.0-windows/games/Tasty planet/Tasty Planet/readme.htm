System Requirements

Minimum:

  • Supported OS: Windows® 98SE/ME/2000/XP/Vista (Tasty Planet does not officially support Windows NT)
  • Processor: Pentium® III 700 MHz
  • RAM: 128 MB
  • Minimum Screen Resolution: 800x600
  • Sound Card: recommended
  • DirectX® Version: DirectX 7.0
  • Hard Drive Space: 20MB

Recommended:

  • Processor: Pentium® III 700 MHz
  • RAM: 256MB recommended for 2000/XP
  • Video Card: 3D Video Card with at least 4 MB Video RAM
  • DirectX® Version: DirectX 8.0 or higher

Installation Instructions

    1. Double-click on the downloaded Tasty Planet installation executable file to begin the installation program.
    2. Follow the instructions in the installation program to install Tasty Planet .
    3. After installation, double-click on the Tasty Planet desktop shortcut, or by using the appropriate shortcut in the Start Menu.
    4. To uninstall Tasty Planet , use the appropriate shortcut in the Windows program menu or by accessing Windows’ Add/Remove Programs control panel.

How To Play

GAME GOALS:

Grow to the size indicated by eating everything smaller than you. The more you eat the bigger you get. You must get to the required size (indicated in the top left corner of the screen) in order to beat the level.

NORMAL:

  • Grow to the needed size by eating everything smaller than you.
  • Levels have a time limit and enemies are aggressive.
  • You can lose the level either by running out of time or by being eaten by certain enemies.

CASUAL:

  • Casual Mode has the same levels as Normal Mode except they are much easier.
  • There is no time limit so you can take as long as you like to beat the levels.
  • Enemies are often less aggressive and do less damage to you.
  • The only way to lose in this mode is by being eaten.

ENDURANCE:

  • Endurance Mode levels take a long time to beat and are quite challenging.
  • Gameplay is similar to Normal Mode except that you will grow much slower.
  • Levels have a time limit and enemies are aggressive.
  • Avoid making any mistakes.
  • You can lose the level either by running out of time or by being eaten by certain enemies.

Cheats

Cheats are fun effects that add to the uniqueness of Tasty Planet. Cheats are activated by beating certain objectives in the game. Some cheats make the game easier and some are strictly for fun. If you do use a cheat that makes the game easier, the game will not save the levels completed.

Cheats are only accessible from the Main Menu/Options/Cheats. You can pick which cheats you would like to try by referring to the list below.

WIDE EYES:

  • The eyes on Grey Goo are extra wide apart.
  • Available after completing Casual Mode.

SMALL EYES:

  • The Grey Goo gets very tiny eyes.
  • Available after completing Casual Mode.

BIG EYES:

  • The Grey Goo gets very large eyes.
  • Available after completing Casual Mode.

CYCLOPSE:

  • The Grey Goo has only one eye.
  • Available after completing Casual Mode.

PERSONIFY:

  • All entities in the game sound like humans when eaten.
  • Available after completing Normal Mode.

FELINIFY:

  • All entities in the game sound like cats when eaten.
  • Available after completing Normal Mode.

RAINBOW GOO:

  • The Grey Goo rapidly changes colors throughout the game.
  • Available after completing Normal Mode.

SUPER FAST:

  • Grey Goo moves faster than normal.
  • Saving the game is not permitted.
  • Available after earning all bronze medals in Normal Mode.

GO THROUGH WALLS:

  • Grey Goo can travel through most objects.
  • Saving the game is not permitted.
  • Available after earning all silver medals in Normal Mode.

INVINSIBLE:

  • Grey Goo is not harmed by anything.
  • Saving the game is not permitted.
  • Available after earning all gold medals in Normal Mode.

GRAVITY WELL:

  • Hold down the mouse button and entities will be drawn towards you.
  • Saving the game is not permitted.
  • Available after completing Endurance Mode.
